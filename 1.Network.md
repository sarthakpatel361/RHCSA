# RHCSA - Basic Network Configuration

## Question

Configure the following:

* Hostname: `servera.lab.example.com`
* IP Address: `172.25.250.10/24`
* Gateway: `172.25.250.254`
* DNS: `172.25.250.254`

---

## Solution

### Configure Network

```bash
nmtui
```

Navigate to:

```text
Edit a connection
```

Configure:

```text
IPv4 Configuration : Manual

Address : 172.25.250.10/24
Gateway : 172.25.250.254
DNS     : 172.25.250.254
```

Save and activate the connection.

---

### Configure Hostname

```bash
hostnamectl set-hostname servera.lab.example.com
```

---

## Verification

### Verify IP Address

```bash
ip addr
```

Expected:

```text
172.25.250.10/24
```

### Verify Gateway

```bash
ip route
```

Expected:

```text
default via 172.25.250.254
```

### Verify DNS

```bash
cat /etc/resolv.conf
```

Expected:

```text
nameserver 172.25.250.254
```

### Verify Hostname

```bash
hostnamectl
```

Expected:

```text
Static hostname: servera.lab.example.com
```

# Basic Troubleshooting

## Check Interface Status

```bash
ip link
```

or

```bash
nmcli device status
```

---

## Check IP Assignment

```bash
ip addr
```

---

## Check Routing

```bash
ip route
```

---

## Test Gateway Connectivity

```bash
ping <gateway-ip>
```

Example:

```bash
ping 172.25.250.254
```

---

## Test DNS Resolution

```bash
ping google.com
```

If this fails but:

```bash
ping 8.8.8.8
```

works, then DNS is likely misconfigured.

## Important RHCSA Commands

```bash
nmtui
hostnamectl
ip addr
ip route
nmcli device status
cat /etc/resolv.conf
```
Method2: Using  nmcli
# 1. Find connection name
nmcli connection show

# 2. Configure IP, Gateway, DNS, Method
nmcli connection modify eth0 ipv4.addresses 172.25.250.10/24
nmcli connection modify eth0 ipv4.gateway 172.25.250.254
nmcli connection modify eth0 ipv4.dns 172.25.250.254
nmcli connection modify eth0 ipv4.method manual

# 3. Apply changes
nmcli connection up eth0

# 4. Set hostname
hostnamectl set-hostname servera.lab.example.com

# 5. Verify
ip addr show eth0
ip route show
cat /etc/resolv.conf
hostname
